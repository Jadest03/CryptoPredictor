# 설명

이 파이썬 프로그램은 ARIMA (자기회귀누적이동평균) 시계열 예측을 사용하여 특정 암호화폐의 성장률을 예측하는 응용 프로그램입니다. <br/>
사용자가 암호화폐의 이름을 입력하면 웹 스크래밍 기술을 통해 가져온 데이터를 머신러닝을 통해 암호화폐 성장률 예측을 제공합니다. <br/>
데이터를 가져오는 초기에는 안정화가 필요합니다. 때문에 예측 값의 변동이 크게 나타날 수 있습니다. <br/>
예측 값이 안정화 될 때 까지 기다려야할 수 있습니다.

# 기능

1. 동적(실시간) 및 정적 데이터 수집: 웹 스크래핑을 사용하여 [Upbit](https://upbit.com/home, "upbit link")에서 지정된 암호화폐의 최신 암호화폐 가격 증가율 데이터와 약 2주간의 가격 증가율 데이터를 가져옵니다. 웹 스크래핑 기술은 Chrome의 가상 드라이버를 이용합니다.
2. ARIMA 시계열 예측: ARIMA 모델을 사용하여 짧은 기간 (3초) 뒤의 암호화폐 성장률을 예측합니다.
3. 사용자 친화적 인터페이스: PyQt5 기반의 그래픽 사용자 인터페이스 (GUI)가 상호 작용 및 시각적으로 매력적인 환경을 제공합니다.
4. 동적 그래플롯: 실시간 및 예측된 백분율 값을 동적 그래프에 표시하여 시각적으로 확인하기 쉽게 합니다.

# 실행방법

프로그램 파일 가져오기:
`git clone https://github.com/jadest03/CoinPredictor.git`

필요한 라이브러리 설치(가상환경 추천):
`pip install -r requirements.txt`

프로그램 실행:
`절대경로/CryptoPredictor/program/thread.py`

# 실행 후

1. 프로그램 실행 시 입력 GUI가 표시됩니다.
2. 예측 하고 싶은 가상화폐의 정확한 이름을 작성합니다.
3. 그래프 GUI를 통해 값을 확인하고 예측 값을 확인합니다
